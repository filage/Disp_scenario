# DispScenario Analyst Frontend

Next.js frontend for DispScenario Analyst.

## Development

```powershell
npm install
npm run dev
```

Open `http://localhost:3000`.

## Checks

```powershell
npm run lint
npm run test
npm run test:e2e
```

The app uses TypeScript, React, Next.js App Router, generated API types, and Playwright for browser coverage.
